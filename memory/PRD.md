|------------|---------|---------|

| **Twitter/X** | `/ads/twitter-pixel.js` | Signup, Purchase, Lead |

- **Archivo:** `/ads/unified-ads-manager.js`
- Envía eventos a TODAS las plataformas simultáneamente
- API simple: `ManoAds.trackSignUp()`, `ManoAds.trackPurchase()`
