# ManoProtect Instaladores - App Android

## Descripción
App móvil profesional para el equipo de instaladores de ManoProtect. Permite recibir órdenes de instalación, confirmar trabajos, actualizar estados y acceder a manuales técnicos.

## Funcionalidades
- Login con credenciales únicas (JWT)
- Recepción de órdenes de instalación
- Confirmación de instalaciones completadas
- Actualización de estado en tiempo real
- Acceso a manuales y guías de instalación
- Notificaciones push de nuevas asignaciones
- Auto-actualización desde Play Store

## Configuración
1. Copiar `google-services.json` del proyecto Firebase
2. Configurar variables de entorno del keystore
3. Ejecutar build: `../scripts/build.sh instaladores`
