# ManoProtect Clientes - App de Seguridad v2.1.0

## Descripción
App móvil para clientes/familias de ManoProtect. Acceso al sistema de alarma, cámaras, botón SOS, historial de eventos y gestión de la cuenta familiar.

## Funcionalidades
- Login con ID de familia + email + contraseña
- Registro de nueva cuenta familiar
- Recuperación de contraseña (email real vía Brevo)
- Panel de seguridad ManoConnect
- Visualización de cámaras CCTV
- Botón SOS de emergencia
- Historial de eventos y alertas
- Auto-actualización desde Play Store

## Configuración
1. Copiar `google-services.json` del proyecto Firebase
2. El keystore `release.keystore` ya está incluido
3. Ejecutar build: `../scripts/build.sh clientes`
