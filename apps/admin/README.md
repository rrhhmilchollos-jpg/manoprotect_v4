# ManoProtect Admin - App Android

## Descripción
App móvil/web de administración para gestores de ManoProtect. Control total del sistema: usuarios, stock, pedidos, instalaciones, logs de auditoría y gestión de versiones.

## Funcionalidades
- Login con doble factor opcional
- Gestión completa de usuarios (comerciales e instaladores)
- Administración de stock central
- Revisión y aprobación de pedidos
- Asignación de instaladores
- Reportes y métricas en tiempo real
- Logs de auditoría detallados
- Gestión de versiones de apps

## Configuración
1. Copiar `google-services.json` del proyecto Firebase
2. Configurar variables de entorno del keystore
3. Ejecutar build: `../scripts/build.sh admin`
