# ManoProtect Shield - Extensión de Chrome

## Descripción
Protección contra estafas en tiempo real mientras navegas. Verifica URLs, emails y teléfonos automáticamente usando las mismas APIs de seguridad que www.manoprotectt.com.

## Características
- ✅ Verificación instantánea de URLs sospechosas
- ✅ Detección de phishing, malware y estafas
- ✅ Menú contextual (clic derecho) para verificar enlaces
- ✅ Alertas visuales en páginas peligrosas
- ✅ Estadísticas de protección
- ✅ APIs EN VIVO: Google Safe Browsing, VirusTotal, AbuseIPDB

## Instalación (Modo Desarrollador)

### 1. Descargar la extensión
- Descarga la carpeta `chrome-extension` completa a tu ordenador

### 2. Abrir Chrome Extensions
- Abre Chrome y ve a: `chrome://extensions/`
- O ve a Menú → Más herramientas → Extensiones

### 3. Activar Modo Desarrollador
- En la esquina superior derecha, activa "Modo desarrollador"

### 4. Cargar extensión
- Haz clic en "Cargar descomprimida"
- Selecciona la carpeta `chrome-extension` que descargaste

### 5. ¡Listo!
- Verás el icono de ManoProtect Shield en tu barra de extensiones
- Haz clic para verificar la página actual
- Clic derecho en cualquier enlace → "Verificar con ManoProtect"

## Uso

### Popup
- Haz clic en el icono de la extensión
- Verás la URL de la página actual
- Haz clic en "Verificar esta página" para análisis completo
- Usa la verificación rápida para comprobar URLs, emails o teléfonos

### Menú Contextual
- Haz clic derecho en cualquier enlace
- Selecciona "Verificar con ManoProtect"
- Verás una notificación con el resultado

### Detección Automática
- La extensión marca automáticamente enlaces sospechosos en rojo
- Pasa el cursor sobre ellos para ver el aviso

## APIs Utilizadas
- Google Safe Browsing (detección de malware/phishing)
- VirusTotal (90+ motores de seguridad)
- AbuseIPDB (reportes de IPs maliciosas)
- AlienVault OTX (inteligencia de amenazas)
- ManoProtect Community (base de datos de reportes)

## Privacidad
- La extensión SOLO envía URLs/datos cuando TÚ haces clic en verificar
- No rastrea tu navegación automáticamente
- No almacena historial de navegación
